# Welcome to FIT Rendezvous

Rendezvous is your entry point over SSH: sign in, pick a **project**, start your **instance**, submit **secrets**, and view the **leaderboard**.

## What to do next

1. Open **Select Project** in the sidebar and choose a competition (or **Deselect** to clear).
2. Use **Project Info** for task text, SSH connection help, instance start/stop, and rankings. The sidebar label uses **green** when your instance is running and **yellow** when it is idle (no color key when no project is selected).
3. **Submit Secret** uses the project selected in the sidebar when you pick one; you can still choose another enrolled project from the dropdown if needed.
4. **Activity log** shows backend messages when you run the full `fit-rendezvous` command.

If something fails (login, instance, or secret check), a message appears at the bottom of the screen or in **Activity log**.
